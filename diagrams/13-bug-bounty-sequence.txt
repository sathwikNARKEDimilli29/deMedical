sequenceDiagram
    actor Researcher as Security Researcher
    participant Platform as Nishkama Platform
    participant BugBounty as Bug Bounty System
    participant Admins as Admin Team
    participant Contracts as Smart Contracts
    
    Note over Researcher,Contracts: Bug Discovery Phase
    
    Researcher->>Platform: Register as security researcher
    Platform->>BugBounty: Create researcher profile
    BugBounty-->>Researcher: Researcher ID + Guidelines
    
    Researcher->>Contracts: Conduct security testing
    Contracts-->>Researcher: Test results
    
    alt Critical Vulnerability Found
        Researcher->>BugBounty: Submit Critical Bug Report
        Note right of Researcher: Severity: CRITICAL<br/>Impact: Fund theft possible
        
        BugBounty->>Admins: Alert! Critical vulnerability
        Admins->>BugBounty: Acknowledge receipt
        
        Admins->>Contracts: Deploy emergency pause
        Contracts-->>Platform: Platform paused
        
        Admins->>BugBounty: Verify vulnerability
        BugBounty->>Researcher: Request additional details
        Researcher->>BugBounty: Provide POC (Proof of Concept)
        
        Admins->>BugBounty: Confirm vulnerability
        BugBounty->>Researcher: Bug approved - Dynamic Reward
        
        Admins->>Contracts: Deploy fix
        Admins->>BugBounty: Mark as resolved
        BugBounty->>Researcher: Transfer reward (ETH/USDC)
        Researcher-->>BugBounty: Reward received
        
    else High Severity Bug
        Researcher->>BugBounty: Submit High Severity Report
        Note right of Researcher: Severity: HIGH<br/>Impact: DOS possible
        
        BugBounty->>Admins: High priority notification
        Admins->>BugBounty: Triage within 24h
        
        Admins->>BugBounty: Confirm bug
        BugBounty->>Researcher: Reward: Dynamic Amount
        
        Admins->>Contracts: Deploy fix in next release
        BugBounty->>Researcher: Transfer reward
        
    else Medium/Low Severity
        Researcher->>BugBounty: Submit Report
        BugBounty->>Admins: Standard notification
        
        Admins->>BugBounty: Review (72h)
        BugBounty->>Researcher: Reward: Dynamic Amount
    end
    
    Note over Researcher,Contracts: Post-Fix Verification
    
    Admins->>Researcher: Request re-testing
    Researcher->>Contracts: Verify fix deployed
    Contracts-->>Researcher: Vulnerability fixed
    Researcher->>BugBounty: Confirm fix
    
    BugBounty->>Platform: Publish responsible disclosure
    Platform-->>Users: Security update notification
    
    Note over Researcher,Contracts: Bug Bounty Leaderboard
    BugBounty->>Researcher: Update leaderboard ranking
    BugBounty->>Platform: Public recognition (optional)
